3. 提权

首先想到使用 suid 提权找到一个属于 root 的有 s 权限的文件。

  •   SUID(Set User ID)，SUID 可以让调用者以文件拥有者的身份运行该文件，所以我们利用 SUID 提权的思路就是运行 root 用户所拥有的 SUID 的文件，那么我们运行该文件的时候就得获得 root 用户的身份了。
