提示

本次考试重要提示：如需计算md5值，请将需要计算的字符串保存为txt，然后拖入winmd5.exe即可

一、(6分)

请下载网页根目录下的www.zip备份文件，审计源码，并利用源码中漏洞获取服务器根目录（而不是网站根目录）文件名为flag_q1的值

二、(10分)

请对该PHP代码进行逆向还原，回答密码明文

需要hex先转ascall码，可以用随波逐流中鸭语Nak解码/hex转bytes。此外key的长度比result短需要填充

三、(10分)

接报一起裸聊诈骗警情后，我侦察员迅速对该服务器布控，流量监测设备发现在时间戳为1700000000000的时候，侦察员通过流量监测设备发现ttttt开头的字符串，请填充ttttt后的32位字符串（提示，请勿在自己手机中安装此APK）

反编译字符串全局搜索ttttt，md5(1700000000000+salt)

public String Md5(long j) {
    String str = j + "**we5r23@34E23EW423Rer@32E42Fklklklk@";
    String str2 = "";
    try {
        MessageDigest instance = MessageDigest.getInstance("MD5");
        instance.update(str.getBytes());
        str2 = bytesToHex(instance.digest());
        Log.d("MD5", str2);
        return str2;
    } catch (NoSuchAlgorithmException e) {
        e.printStackTrace();
        return str2;
    }
}

四、(6分)

情通过该网页读取服务器根目录（而不是网站根目录）文件名为flag_q4的值

这个特征可以用文件协议ssrf漏洞

song{song_ssrf_file}

五、(10分)

请分析该流量包，截取其中rar压缩包，并回答该压缩包密码

解析流量包，蚁剑的流量，base64解密需要删除前面两个字符

六、(6分)

发现一违法四方支付网站，请回答该网站数据库名

sqlmap一把梭

七、(10分)

获取到kk园区后台权限，查看其源码，发现该网站用户名、密码的加密方式，请根据该加密代码，计算出密码为”123456”的密码密文，从而在数据库中添加后门账号

八、(10分)

逆向该程序密码

九、(16分)

十、(16分)

利用该web初始化安装程序，获取服务器根目录（而不是网站根目录）下flag_q10的值
