QEMU 启动!

-s：-gdb tcp::1234 的缩写，QEMU 监听在 TCP 端口 1234，等待 gdb 的连接。
-S：在启动时冻结 CPU，等待 gdb 输入 c 时继续执行。
-kernel：指定内核。
-initrd：指定 initramfs。
nographic：禁用图形输出并将串行 I/O 重定向到控制台。
-append “console=ttyS0：所有内核输出到 ttyS0 串行控制台，并打印到终端。
